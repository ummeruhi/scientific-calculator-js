# 🧮 Scientific Calculator (JavaScript)

A modern **Scientific Calculator Web App** built using **HTML, CSS, and JavaScript** with an interactive UI and advanced mathematical functionality.

This project demonstrates **DOM manipulation, event handling, keyboard shortcuts, UI animations, and mathematical expression evaluation** using pure JavaScript.

---

# 🚀 Features

✔ Compact modern calculator UI  
✔ Dark / Light theme toggle  
✔ Degree / Radian mode switching  
✔ 2nd function mode (inverse trig etc.)  
✔ Memory functions (MC, MR, M+, M-)  
✔ Scientific operations  
✔ Keyboard shortcuts support  
✔ Expression history panel  
✔ Smart error handling  
✔ Ripple button animation  
✔ Clipboard copy / paste support  

---

# 🧠 Supported Operations

Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

Scientific Functions
- sin(x)
- cos(x)
- tan(x)
- asin(x)
- acos(x)
- atan(x)
- log(x)
- ln(x)
- sqrt(x)
- factorial (n!)
- power (xʸ)
- square (x²)
- inverse (1/x)
- percentage (%)

Constants
- π (pi)
- e (Euler’s number)
- ANS (previous result)
